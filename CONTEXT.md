# IELTS Academic Vocabulary Deck

IELTS / Academic English Anki deck builder, notes DB, and scraper pipeline.

## Language

**EAVM Note Type**:
The Anki note type for the English Academic Vocabulary Model, generated from EAVM styling and templates.
_Avoid_: Card type, template model

**Idiom Label**:
An inline visual label prepended next to a definition to indicate that the sense is an idiom.
_Avoid_: Idiom marker, idiom tag

**Register Tag**:
A visual chip prepended inline to definitions to show register, usage restrictions, or subject contexts.
_Avoid_: Register chip, usage tag

**Feature Tag**:
A prominent standalone badge displayed at the bottom of the card content representing card-level features like phrasal verbs.
_Avoid_: Bottom tag, feature button
